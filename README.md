# An-lisis-de-datos-socioecon-micos-Huila-y-Neiva-Colombia-
Este repositorio presenta un análisis demográfico integral del departamento del Huila y el municipio de Neiva, desarrollado en el lenguaje de programación R. El objetivo principal del proyecto es caracterizar la estructura y dinámica poblacional del territorio analizado mediante el cálculo y estudio de diversos indicadores demográficos.
